### Co-ordinate System Conversion


| Given | Transform                         |
| ----- | --------------------------------- |
| $x$   | $\rho = \sqrt{x^2+y^2}$           |
| $y$   | $\theta = \tan^{-1}(\frac{y}{x})$ |
| $z$   | $z$                               |

#### Cartesian Co-ordinate System to Cylindrical

| To  | From                |
| --- | ------------------- |
| $x$ | $\rho \cos(\theta)$ |
| $y$ | $\rho \sin(\theta)$ |
| $z$ | $z$                 |
